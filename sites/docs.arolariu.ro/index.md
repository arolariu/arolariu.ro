# docs.arolariu.ro

Welcome to the unified documentation portal for the arolariu.ro monorepo.

## Start here

- Monorepo guides & RFCs: [Monorepo Docs](articles/monorepo/index.md)
- TypeScript reference (TSDoc/JSDoc): [TypeScript API](articles/ts-api/index.md)
- .NET reference (XML docs): [API Documentation](api/index.md)

## What you’ll find

- **Monorepo Docs**: engineering guides and architectural RFCs (imported from the repo’s `docs/` folder).
- **General Articles**: curated, human-written articles that don’t fit the guide/RFC structure.
- **TypeScript API**: generated reference from TSDoc/JSDoc for `@arolariu/components` and selected website modules.
- **API Documentation**: generated reference from C# XML docs.
