# docs.arolariu.ro (DocFX)

This app builds and serves the documentation website for the arolariu.ro monorepo.

## Golden rule

- `sites/docs.arolariu.ro/_site/` is generated output. Never edit it directly.

## Source inputs

- Prose docs: `docs/` (imported into `articles/monorepo/` during DocFX build)
- Hand-written site pages: `sites/docs.arolariu.ro/` (e.g., `index.md`, `articles/**`)
- TypeScript API reference: generated by TypeDoc into `sites/docs.arolariu.ro/articles/ts-api/latest/**/reference/`

## Commands

- Build: `npm run build:docs`
- Dev server: `npm run dev:docs`
- Clean: `nx run docs:clean`
